NF Subpages adds pages and fieldset elements.

== Description ==

NF Subpages adds pages and fieldset elements to the 'Layout' Elements' block. It also includes labels and custom classes for elements to allow for more interesting layouts and styling.
Subpages works only with javascript, otherwise it displays the pages with fieldsets.

This plugin requires the Ninja Forms plugin.

== Installation ==

1. Upload 'nf-subpages.php' and all subfolders to the '/wp-content/plugins/'' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
